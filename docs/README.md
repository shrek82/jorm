## 特性

JORM 提供以下主要特性：

- ✅ **轻量高效** - 核心依赖最少，代码精简，性能优化
- ✅ **链式操作** - 流畅的 API 设计，支持链式调用
- ✅ **类型安全** - 基于反射的动态类型处理
- ✅ **多数据库支持** - MySQL, PostgreSQL, SQLite, Oracle, SQL Server
- ✅ **事务管理** - 函数式事务，自动提交/回滚
- ✅ **钩子函数** - Before/After 操作钩子
- ✅ **连接池** - 内置连接池管理，支持重试机制
- ✅ **自动迁移** - 基于模型的表结构自动创建和更新
- ✅ **关联预加载** - 支持各种关联关系
- ✅ **数据验证** - 内置数据验证器
- ✅ **Context 支持** - 支持超时控制和操作取消
- ✅ **日志系统** - 可配置的日志系统

## 代码示例

所有文档都包含完整的代码示例，可以直接复制使用。

### 基本查询

```go
var users []User
err := db.Model(&User{}).Where("age > ?", 18).Find(&users)
```

### 插入数据

```go
user := &User{Name: "Alice", Email: "alice@example.com"}
id, err := db.Model(user).Insert(user)
```

### 事务处理

```go
err := db.Transaction(func(tx *core.Tx) error {
    tx.Model(&User{}).Insert(user)
    tx.Model(&Order{}).Insert(order)
    return nil
})
```
