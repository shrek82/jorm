## 学习路径

### 初学者

1. 快速开始 → 了解基本概念
2. 数据库连接 → 建立数据库连接
3. 模型定义 → 定义数据模型
4. 查询操作 → 学习查询数据
5. 插入操作 → 学习插入数据

### 中级开发者

1. 更新操作 → 学习更新数据
2. 删除操作 → 学习删除数据
3. 事务处理 → 保证数据一致性
4. 数据验证 → 验证数据有效性

### 高级开发者

1. 钩子函数 → 实现自定义逻辑
2. 关联关系 → 处理复杂关联
3. 迁移管理 → 管理 schema 变更
4. 性能优化 → 优化应用性能
